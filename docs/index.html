Neptune's Pride Agent

Neptune's Pride is a 4x space conquest game written by Jay Kyburz of Iron Helmet Games. To succeed at the game, players have to excel at diplomacy and calculation.

Neptune's Pride Agent (or NPA) is a fan-built extension that injects javascript into the game to speed up in-game decision making and eliminate many tedious calculation or UI navigation tasks. NPA consists of:

  • A heads-up display (HUD) for battle and scanning info
  • Globally modified timestamps for easier planning
  • New linkable elements for in-game messaging
  • Multiple report screens for review or sharing
  • Extensive API key support for easier alliance management

NPA can be installed on Chrome, Firefox, and Safari. The easiest way is to install the Chrome extension from the Chrome Web Store or the Firefox extension from addons.mozilla.org. It is possible to install on Android and iOS but the process is a little bit more complicated than on desktop.
